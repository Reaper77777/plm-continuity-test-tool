# PLM Continuity Test Tool

## Overview
Excel-based continuity and isolation verification tool designed to standardize connector testing workflows and improve manufacturing quality documentation.

## Features
- Continuity verification tracking
- Isolation testing
- Pass/Fail dropdowns
- Automated conditional formatting
- Technician sign-off
- Failure logging
- Quality reporting

## Tools Used
- Microsoft Excel
- Data Validation
- Conditional Formatting
- Electrical Test Procedures

## Skills Demonstrated
- Test Engineering
- Quality Assurance
- Process Improvement
- Electrical Testing
- Technical Documentation
